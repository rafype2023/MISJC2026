Configuración de GitHub

Conecta el editor con tu cuenta de GitHub para que las actualizaciones se publiquen automáticamente en internet.

🔑 Seguridad: Este token se guarda únicamente en el almacenamiento local de tu propio navegador (`localStorage`). Ningún servidor externo tiene acceso a él.

¿Cómo crear un token?
1. Haz clic aquí para ir a GitHub: Crear Token de Acceso.
2. Activa la casilla repo (permite escribir en el repositorio).
3. Haz clic en "Generate token" al final de la página de GitHub.
4. Copia el token generado y pégalo en el campo superior.
🚀
¿Cómo publicar los cambios?
  1. Completa la pestaña ⚙️ Conexión GitHub con tu token (solo debes hacerlo la primera vez).
  2. Haz clic en el botón rojo superior Guardar Cambios.
  3. ¡Listo! El editor actualizará el archivo en GitHub automáticamente y Render.com compilará y actualizará el sitio en 1 o 2 minutos.

¡Guardando datos del Diario!

El navegador descargará un archivo llamado diario_data.json.

1
Guarda el archivo en tu computadora.
2
Mueve o copia ese archivo a la carpeta del proyecto MisJC2026 travel portfolio para reemplazar el anterior.
3
Ejecuta el script actualizar.command para subir tus cambios a internet.
